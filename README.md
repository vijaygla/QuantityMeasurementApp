# 📘 Quantity Measurement System  

### (UC1 – UC18 Implementation)

A **multi-layer enterprise-level .NET application** designed to perform **quantity measurement operations** such as:

- Comparison  
- Unit Conversion  
- Arithmetic Operations  

Across measurement categories:

- Length  
- Weight  
- Volume  
- Temperature  

---

## 🚀 Project Evolution (UC1 → UC18)

This project is built incrementally using **Use Case driven development**:

| Use Case | Description |
|----------|------------|
| UC1      | Basic equality check for quantities |
| UC2      | Handle different units (feet, inch, etc.) |
| UC3      | Unit conversion support |
| UC4      | Addition of quantities |
| UC5      | Subtraction support |
| UC6      | Multiplication / scaling |
| UC7      | Temperature conversion |
| UC8      | Exception handling |
| UC9      | OOP principles (Encapsulation, Abstraction) |
| UC10     | Polymorphism & Operator Overloading |
| UC11     | Clean code & refactoring |
| UC12     | Layered Architecture introduction |
| UC13     | Repository Pattern implementation |
| UC14     | Database integration (SQL Server) |
| UC15     | Logging & Exception tracking |
| UC16     | Unit Testing |
| UC17     | REST API (ASP.NET Core MVC) |
| UC18     | Authentication + User-based access system |
